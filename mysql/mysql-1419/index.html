MySQL 创建触发器的时候报 1419 错误 (1419 - You do not have the SUPER privilege and binary logging is enabled)

March 29, 2017 249

MySQL 创建触发器的时候报错：

解决方法：

  1. 用 root 用户登录：mysql -u root -p
  2. 设置参数 log_bin_trust_function_creators 为 1：set global log_bin_trust_function_creators = 1;

再次尝试创建触发器，成功！(同样，在删除触发器报该错误的时候也是如此处理）
